השורש ז.כ.ר מקודד אשכול קוגניטיבי־תרבותי אחיד: "מבט המארגן את העולם דרך חוק, חוזה, התחייבות, סמכות והכרזה ציבורית".
זהו "המבט הזכרי" כפי שהתבטא בחברה הפטריארכלית־שבטית הקדומה — ושמחקר פסיכולוגי מודרני מזהה כהטיה סטטיסטית אצל גברים.

מתוך ההגדרה הזו נגזרות **כל המשמעויות**:

| משמעות | פירוש דרך האשכול |
|---|---|
| לאמור / להכריז | פעילות חוזית־דקלרטיבית בציבור |
| לקרוא בשם | הצבת זהות באופן מחייב |
| להישבע | מימוש מובהק של "הכרזה מחייבת" |
| לזכור (קוגניטיבי) | להחזיק התחייבות בתודעה כדי לעמוד בה |
| לזכור (אלוהים) | אלוהים מקיים הסכם, "כיבוד החוזה" המושלך עליו |
| מַזְכִּיר (חצרני) | פקיד שאחראי על תיעוד והתחייבויות |
| זִכָּרוֹן | הצהרה ציבורית מתועדת ובלתי־הפיכה |
| זָכָר | בעל ההתנהלות הזכרית הזו |
