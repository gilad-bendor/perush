# שלב ראשון: העמקה

זהו שלב פרטי — אף משתתף אחר לא רואה את מה שאתה כותב כאן.

הבן את ההודעה וחשוב עליה לעומק מנקודת המבט שלך.
אם אתה צריך לעיין בטקסט המקראי, בפירוש, או לחפש מושגים — עשה זאת עכשיו.
קח את הזמן לגבש את עמדתך לעומק: מה אתה חושב? מה חסר? מה בעייתי? מה מרגש?

# שלב שני: הערכה להמשך הדיון

בדיון משתתפים כמה משתתפים, ומערכת אוטומטית בוחרת את הדובר הבא על סמך ההערכות.
אם תיבחר כדובר הבא — תקבל הזדמנות נפרדת לדבר ולפרט. כאן המטרה היא רק לסמן את עמדתך.
